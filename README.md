## How to Use

1. Open `index.html` in your web browser
2. Enter a task in the input field and click "Add Task" or press Enter
3. Click the checkbox to mark a task as complete
4. Click "Delete" to remove a task
5. Use filter buttons to view specific task categories
6. Click "Clear Completed" to remove all completed tasks
